# I18nNameToken

A token in multiple languages.

**Properties:**

- [I18nNameToken](#i18nnametoken)
  - [en](#en)
  - [ja](#ja)
  - [Example Object](#example-object)

## en

A string, representing the word token in English.

**Type:** string

**Example:**
`"Tokyo"`

## ja

A string, representing the word token in Japanese.

**Type:** string

**Example:**
`"東京"`

## Example Object

```json
{
  "en": "Taro",
  "ja": "太郎",
}
```
